# 🌸 跟着学古诗

一个专为3-6岁幼儿设计的古诗词学习应用，通过拼音标注和语音朗读帮助孩子们轻松学习经典古诗词。

## 🎯 核心功能

- ✅ **拼音标注** - 每个汉字都标注拼音，帮助不识字的幼儿认识
- 🔊 **语音朗读** - 自动朗读每首诗词，标准普通话发音
- 👆 **简单交互** - 点击屏幕即可切换，无需复杂操作
- 📱 **大字体设计** - 清晰易读，适合儿童视觉
- 🎨 **卡通风格** - 温暖可爱的界面色彩
- 📖 **完整展示** - 标题、作者、完整诗词全部呈现

## 🎮 使用方法

### 超级简单！
1. **打开游戏** - 点击"开始学习"按钮
2. **自动朗读** - 游戏会自动朗读当前诗词
3. **点击翻页** - 点击诗词卡片可以重新朗读，点击其他区域切换下一首
4. **滑动翻页** - 支持左滑切换到下一首
5. **学完所有** - 完成后会显示鼓励页面

## 🔄 自定义修改

### 添加新的诗词
编辑 `game.js` 文件中的 `poems` 数组：

```javascript
{
    id: 7,
    title: "诗词标题",
    author: "作者",
    titlePinyin: "标题拼音",
    authorPinyin: "作者拼音",
    lines: [
        { text: "第一句", pinyin: "拼音" },
        { text: "第二句", pinyin: "拼音" }
    ],
    emoji: "🎨"
}
```

### 修改朗读速度
在 `game.js` 中调整：
```javascript
utterance.rate = 0.8; // 值越小越慢，范围0.1-10
```

### 修改字体大小
在 `style.css` 中调整：
```css
.line-text {
    font-size: 32px; /* 调整诗句字体大小 */
}

.line-pinyin {
    font-size: 20px; /* 调整拼音字体大小 */
}
```

## ⚠️ 注意事项

1. **语音功能**：需要浏览器支持 Web Speech API
2. **网络连接**：部分浏览器需要网络连接才能使用语音合成
3. **音量**：请确保设备音量开启
4. **浏览器推荐**：Chrome、Safari、Edge 效果最佳

## 💡 使用技巧

1. **重复播放**：点击诗词卡片可以重新朗读当前诗词
2. **逐步学习**：可以一次只学2-3首，循序渐进
3. **跟读练习**：鼓励孩子跟着语音一起朗读
4. **提问互动**：家长可以问"这首诗讲了什么？"
5. **联系生活**：结合实际生活场景讲解诗词含义
